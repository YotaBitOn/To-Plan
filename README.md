# To-Plan

To-Plan is and advanced **to-do list** application which helps you organise your time, plan your day and track & analyse your progress
<img width="1109" height="818" alt="image" src="https://github.com/user-attachments/assets/bc05a784-bc22-4d87-821f-fe0116775319" />

## With it you can
  - Create, edit and remove tasks
  - Make tasks repeatable instead of creating new one every time
  - Set your task:
      -Description
      -Difficulty
      -Category
      -Date and time of apperaring
      -Due time
      -Task steps

## Stack:
```
  - Pyside, PyQT
  - Qt Creator
  - SqlLite
  - Smaller libraries: nanoid, dotenv, collections, webbrowser, datetime, sys, os 
```

## Use instructions:
```
  1. Click on "+" button on the bottom of main tab to create task
  2. On popped up window configure your task
  3. Click "Submit button" on the bottom of popped up window
  4. After this you will see that main tab was splitted into 2 parts:
     - Current date task list on the rihgt
     - Current task info on the left
    
  5.You can also use navigation bar to see your stats or go to settings

  ```
